# БЫСТРЫЙ СТАРТ - Сборка EXE

## Шаг 1: Подготовка
1. Убедитесь, что установлен Python 3.8+
2. Откройте командную строку в папке проекта

## Шаг 2: Автоматическая сборка
```bash
build.bat
```

Готово! EXE будет в папке `dist\NodeBridge.exe`

## Шаг 3: Финализация
1. Скопируйте ваш файл `server.js` в папку `dist\` рядом с `NodeBridge.exe`
2. Запустите `NodeBridge.exe`

---

## Альтернатива: Ручная сборка

```bash
# 1. Создать виртуальное окружение
python -m venv venv

# 2. Активировать
venv\Scripts\activate

# 3. Установить зависимости
pip install -r requirements.txt

# 4. Собрать EXE
pyinstaller --clean NodeBridge.spec
```

---

## Важные заметки

✅ **Обязательно**: `server.js` должен быть рядом с EXE
✅ **Обязательно**: Node.js должен быть установлен
✅ **Обязательно**: Google Chrome должен быть установлен

❌ **Не включайте** `history.json` в сборку (создается автоматически)
